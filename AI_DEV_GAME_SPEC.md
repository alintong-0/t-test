# AI 驱动开发技术规格说明书（简版）

## 1. 目标
构建一个可在浏览器直接运行的轻量游戏，核心强调“AI 导演系统”对节奏和难度的实时调节。

## 2. 玩法概述
- 类型：2D 俯视躲避生存。
- 玩家控制飞船左右移动，躲避下落障碍。
- 随时间推进，分数持续增长。

## 3. AI 导演系统
- 输入信号：
  - 玩家最近 8 秒受击次数。
  - 最近 8 秒险些碰撞次数（Near Miss）。
  - 玩家当前连击（持续无伤时间）。
- 输出：
  - 障碍物生成速率。
  - 障碍物下落速度。
  - 特殊奖励（护盾）掉落概率。
- 策略目标：
  - 玩家表现强：提高难度（更快、更密）。
  - 玩家表现弱：降低难度并提高恢复资源。

## 4. 技术约束
- 纯前端：HTML + CSS + Vanilla JS。
- 不依赖构建工具。
- 支持键盘与移动端触控。

## 5. 可玩性验收标准
- 打开 `index.html` 可直接运行。
- 平均帧率稳定，操作无明显输入延迟。
- 失败后可“一键重开”。
